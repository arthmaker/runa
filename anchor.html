Tools Anchor & Link Gambar

Buat daftar anchor text, pisahkan link-nya, dan generate daftar link gambar hanya dari daftar judul.

← Kembali ke Generator

1) Input

Judul
Dipakai sebagai anchor text dan sebagai bahan slug.
Hasil link: baseUrl + slug + suffix
Contoh: .html / / / kosong
Hasil gambar: imgBase + slug + imgExt
Contoh: .webp, .jpg, .png
Masukkan judul, lalu klik Generate Output.

2) Output

Copy-paste
Output link gambar ini bisa langsung kamu tempel ke kolom GAMBAR di halaman generator.

Tips

Praktis
Cara pakai cepat
  • Tempel daftar judul → klik Generate Output.
  • Anchor Text = judul (dibersihkan spasi), Link = base URL + slug.
  • Jika link kamu model folder (tanpa .html), isi Suffix dengan /.
