Tools Anchor, Link & Gambar

Generate anchor text dari daftar judul, ambil link (href), dan susun link gambar berurutan.

← Kembali ke Generator

1) Input

Judul
Dipakai sebagai anchor text dan sebagai bahan slug.
Anchor: <a href="baseUrl + slug + .html">...</a>
Default: .html
Limit utama slug. Kata terakhir boleh selesai sampai +12 karakter.
Masukkan judul, lalu klik Generate Output.

2) Output

Copy-paste

3) Link Gambar

3 Box
Pastikan diakhiri / (folder).
Nomor 1–10 akan ditambahkan otomatis.
Contoh: .webp, .jpg, .png
Isi 3 box → klik Generate.

Praktis

  • Tempel daftar judul → klik Generate Anchor Text.
